---
description: CHANGELOG.md 업데이트 (최근 커밋 기반)
---

docs/CHANGELOG.md를 업데이트해줘.

1. `git log --oneline -10` 확인
2. docs/CHANGELOG.md 현재 상태 확인
3. 새 커밋들을 분류 (feat/fix/docs/refactor/chore)
4. 오늘 날짜로 섹션 추가
5. 기존 내용 유지하면서 상단에 삽입

형식:
```markdown
## [날짜]

### Added
- 새 기능

### Changed
- 변경 사항

### Fixed
- 버그 수정
```
