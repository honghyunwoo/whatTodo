# 백업/복원 수동 테스트 메모

## 기능 개요
- `utils/backup.ts`에서 다이어리(JOURNAL), 할 일(TASKS), 학습 진행(LEARN_PROGRESS), SRS, 설정 키를 JSON으로 묶어 내보내고 붙여넣기로 복원합니다.
- `app/settings.tsx` 설정 화면에서 "백업 내보내기" / "백업 불러오기" 버튼을 제공합니다.

## 수동 테스트 절차
1. 앱을 실행하고 설정(우상단 톱니) → 백업/복원 화면으로 이동합니다.
2. 더미 데이터(다이어리/할 일/학습/SRS)를 만든 뒤 **백업 내보내기**를 눌러 공유 시트를 확인하고 JSON이 입력창에 채워지는지 확인합니다.
3. 앱 데이터를 일부 수정/삭제한 뒤, 이전 단계에서 복사한 JSON을 입력창에 붙여넣고 **백업 불러오기**를 누릅니다.
4. 성공 얼럿을 확인하고 탭 전환 또는 앱 재시작 후 데이터가 백업 시점과 동일하게 복원되었는지 확인합니다.
5. 잘못된 JSON을 붙여넣었을 때 에러 얼럿이 뜨는지 확인합니다.
