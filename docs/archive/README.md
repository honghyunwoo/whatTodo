## 📝 보관 정책

### 왜 삭제하지 않나요?

1. **기록 보존**: 과거 분석 내용 참고
2. **실수 학습**: 어떤 정보가 잘못되었는지 기록
3. **비교**: 시간 경과에 따른 변화 추적

### 언제 archive로 이동하나요?

- 새로운 분석/보고서가 작성될 때
- 정보가 outdated되었을 때
- 파일명에 날짜 폴더로 정리: `archive/YYYY-MM-DD/`

### 언제 삭제하나요?

- 6개월 이상 지난 문서
- 더 이상 참고 가치가 없을 때
