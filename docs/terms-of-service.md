## 1. 서비스 개요

### 1.1 앱 소개

"What To Do?" (이하 "앱")는 할 일 관리와 영어 학습을 결합한 모바일 애플리케이션입니다.

### 1.2 주요 기능

- Todo 관리 (생성, 수정, 삭제, 완료)
- 영어 학습 (어휘, 문법, 듣기, 읽기, 말하기, 쓰기)
- 학습 진행 추적 (SRS 복습 시스템)
- 개인 메모 및 일기
- 캘린더 기반 기록 관리
- 데이터 백업/복원

## 3. 서비스 이용

### 3.1 오프라인 사용

- 본 앱은 완전 오프라인으로 작동합니다
- 인터넷 연결 없이 모든 기능을 사용할 수 있습니다

### 3.2 데이터 저장

- 모든 데이터는 사용자의 기기에만 저장됩니다

- 데이터 백업은 사용자의 책임입니다

### 3.3 학습 콘텐츠

- 학습 콘텐츠는 앱 내에 포함되어 있습니다
- 콘텐츠는 교육 목적으로만 제공됩니다

- 앱을 본래 목적(할 일 관리, 영어 학습)에 맞게 사용
