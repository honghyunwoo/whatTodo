# Sound Effects

이 폴더에 사운드 파일을 추가하면 앱에서 효과음이 재생됩니다.
사운드 파일이 없어도 앱은 정상 작동합니다 (햅틱 피드백만 제공).

## 무료 사운드 효과 사이트

- [Freesound](https://freesound.org/) - CC0 라이선스 효과음
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/) - 무료 상업적 사용 가능
- [Zapsplat](https://www.zapsplat.com/) - 무료 회원가입 후 다운로드
- [Mixkit](https://mixkit.co/free-sound-effects/) - 무료 효과음

## 파일 형식

- 형식: MP3 (권장) 또는 WAV
- 샘플레이트: 44100Hz
- 비트레이트: 128-192kbps
- 볼륨: 정규화 (-3dB ~ 0dB)

## 추가 방법

1. 위 사이트에서 적절한 효과음 다운로드
2. 파일명을 위 표에 맞게 변경
3. 이 폴더에 저장
4. 앱 재시작

파일이 추가되면 자동으로 효과음이 활성화됩니다.
